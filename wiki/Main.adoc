== Istallation

. Setup your OS-preferred package system repo following link:https://github.com/igagis/pravila/blob/master/EnableRepo.adoc[this manual]
. Install package
+
- **deb** (Linux): **libsvgren-dev**
- **homebrew** (MacOS X): **libsvgren**
- **Android**: **io.github.igagis:svgren**
- **cocoapods** (iOS): **svgren**
- **Msys2** (Windows): **mingw-w64-i686-svgren**, **mingw-w64-x86_64-svgren**
- **Nuget** (Windows, Visual Studio): **libsvgren**
